# Nuxt + Express

> Nuxt application using Express as API.

## Build Setup (Nuxt)

``` bash
# Dependencies
$ npm install

# Production build
$ npm start
```

Then, access it at http://localhost:3000/

## Build Setup (Express/ API)

``` bash
# Dependencies
$ npm install

# Production build
$ npm start
```

Then, access it at http://localhost:3030/

## Notes

1. For this approach, you must run these two apps concurrently.
