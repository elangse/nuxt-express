# Nuxt + Express

> Nuxt application using Express as API.

## Build Setup (Nuxt)

``` bash
# Dependencies
$ npm install

# Production build
$ npm start
```

Then, access it at http://localhost:3000/

## Build Setup (Express/ API)

``` bash
# Dependencies
$ npm install

# Production build
$ npm start
```

Then, access it at http://localhost:3030/


## Dependencies

* [Nuxt.js](https://nuxtjs.org/)
* [Express](http://expressjs.com/)
